Importent FAQ

What is the Document Object Model (DOM)?

The DOM is basically a way to represent a webpage's structure as a tree of elements. It lets JavaScript interact with the HTML, so you can change things like text, styles, or even add new elements.

How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById, document.querySelector, or document.getElementsByClassName. These let you grab specific elements based on their ID, class, or tag and do stuff with them like changing text or styles.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when you attach an event listener to a parent element instead of individual child elements. It works by taking advantage of event bubbling, where events fired on child elements propagate up to their parent. It's useful because it reduces the number of event listeners needed, making your code more efficient, and works for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can change an element's attributes using setAttribute() or get them with getAttribute(). For example, you could set an image's src attribute. To modify styles, you can directly change the style property, like element.style.backgroundColor = "blue";, or manage CSS classes with classList.add() or classList.remove(). This lets you update how elements look or act on a webpage.
